# GoGreen Recycling System 🌱

A web-based recycling system using:
- Apache
- PHP
- MySQL
- JavaScript

Features:
- Recycling point system
- Quiz rewards
- Smart map
- Redeem system
- Admin dashboard
- Export to Excel

How to run:
1. Install XAMPP
2. Put project in htdocs
3. Import database
4. Run localhost/gogreen
